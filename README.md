#SCS-C03 Study Hub

Interactive beta study hub for the **AWS Certified Security - Specialty SCS-C03** certification.

This project was created to make AWS Security studies more practical, visual, and closer to real Cloud Security work.

## What is included

* Guided lessons by exam domain
* Real-world security scenarios
* Exam-style questions
* Flashcards
* AWS service comparisons
* Simulated console interpretation
* Decision engine
* Service deep dives
* Flow maps
* Quick cheatsheets
* Favorites and personal notes
* Local progress tracking
* English and Portuguese support
* Bonus AI Security module

## Exam domains

The app covers the 6 official AWS Security Specialty study areas:

* D1 - Detection and Monitoring
* D2 - Incident Response
* D3 - Infrastructure Security
* D4 - Identity and Access Management
* D5 - Data Protection
* D6 - Security Governance and Foundations

It also includes a bonus module:

* D7 - AI Security

The D7 module is not part of the official AWS exam. It was added as complementary content because Cloud Security is increasingly connected to AI workloads, LLM applications, RAG, agents, machine learning pipelines, data exposure, and AI governance.

## Main goal

The goal is to help learners review AWS Security concepts, understand common exam traps, compare similar services, and practice decision-making through real scenarios.

This is not just a checklist.
The idea is to connect theory, exam preparation, and practical Cloud Security reasoning.
